Mod Command Reference

For the mod squad only 🍄 · updated July 16, 2026

360 Camera — Pan & Spin

!cam frontFace forward (0°) and level out the camera
!cam backFace behind (180°) and level out
!cam left
!cam right		Pan to -90° / 90° and level out
!cam 135Pan to any exact angle (-180 to 180)
!spinFull 360° spin, ends where it started (keeps tilt & zoom)

New followers trigger an automatic spin — no command needed.

Tilt (up / down)

!cam upLook straight up
!cam downLook straight down
!cam levelReturn to level — fixes tilt AND any crooked roll
!cam tilt 45Tilt to an exact angle, -90 (down) to 90 (up)

Zoom

!zoom 80Zoom to an exact level. Lower = more zoomed in. Chat range: 60–120
!zoom in
!zoom out		Step the zoom tighter / wider
!zoom resetBack to the normal default view

All zoom commands also work as !cam zoom … — same thing.

Roll (the fun ones)

!barrelrollFull 360° barrel roll, lands exactly where it started
!cam roll 15Hold a tilted "dutch angle", -180 to 180. !cam roll 0 straightens out

Delivery Dashbar (delivery streams)

!adddelAdd a completed delivery to the on-screen tracker
!undodelUndo the last added delivery (mistake fix)
!cleardelClear the delivery list
!resetdelFull reset of the tracker — counter, earnings, and goal

Good to know

  • Camera commands run one at a time in a queue — if nothing happens right away, a move is mid-animation. It will run; no need to spam.
  • Camera stuck or crooked? !cam front then !zoom reset fixes almost everything.
  • Direction presets automatically re-level the camera, and zoom stays where it's set until changed or reset.
  • Chat zoom is clamped to a safe range on purpose — nobody can mush-zoom or fisheye the stream. It's working as intended.
  • If Streamer.bot restarts mid-stream, the camera assumes front / level / default zoom — one !cam front + !zoom reset re-syncs it.
